Content:
美团研究院报告：户外运动“大众化”加速，消费人次超8亿，女性消费者占比54.6%
从城市绿道上络绎不绝的骑行者，到网球、攀岩成打工人热衷的体验，再到溯溪、徒步、露营变为新的社交方式……丰富多元的运动体验，正在走进更多人的日常生活。

北京体育大学体育休闲与旅游学院、中国体育发展研究院、全民健身研究院与美团研究院联合发布的《"小众运动体验"走向"大众生活方式"——户外运动消费发展及趋势报告》(以下简称《报告》)显示，2025年我国户外运动消费人次超8亿，消费总额超1万亿元。户外运动从小众圈层走向全民体验，为消费者提供即时反馈的运动"多巴胺"，成为"情价比消费"重要组成部分。赛事经济的蓬勃发展，搭建吃喝玩乐一站式服务消费体系，正在成为城市消费新增量。

从小众圈层到全民体验，体验普惠激活户外运动消费新动能

"周末打算约朋友先去骑行，然后再去拉伸放松，打工人的回血体验。"上海李女士的分享正在成为很多人的生活体验……运动体验经济也从小众圈层的爱好，走向大众日常生活的标配选项。

美团研究院数据显示，从消费人群来看，20至44岁是消费主力，女性消费者占比达54.6%，已成为户外消费强劲驱动力(6.120, -0.45, -6.85%)。而在消费城市上，三线城市消费占比达23.4%，位列各级城市之首，运动体验正从一二线城市加速下沉。同时，随着深度体验玩法的兴起，"跨城找运动体验"成为潮流趋势，异地消费者占比近78%。


《报告》显示，从具体运动热门程度来看，冰雪运动中，滑雪消费占比达91%，而冰雪运动正从季节性体验升级为冬季生活方式；陆地运动中卡丁车、露营、蹦床消费占比分别为41.8%、24.3%和14.2%；水上运动中，漂流消费占比高达77.9%；航空运动以高空体验的稀缺感成为潮流玩法新热点，滑翔伞和跳伞消费占比分别达34.5%和33.8%，"尝鲜经济"成热门户外体验；匹克球等小球运动快速兴起，活跃用户超3000万，低竞技门槛叠加轻社交属性，催生出朋友局、家庭局、商务休闲局等多元消费场景。

无论是入门型垂钓、骑行、露营等低门槛项目，还是网球、滑雪等专业进阶运动，运动体验多元生态蓬勃发展。通过美团线上平台的丰富供给，曾经需要专业装备和长途奔赴才能完成的体验，如今在城市周边，用可控的预算在周末就能实现。正是这种"体验普惠"，共同撑起这个超万亿规模的市场。

运动体验经济从赛道到商圈，赛事带动城市消费"活"起来

"为一场比赛，奔赴一座城"成为新风尚，随着参赛人群的跨城流动，赛事举办城市的异地消费需求增长明显，除了赛事相关的运动装备购买、专业训练以外，还衍生出异地户外运动、餐饮住宿、文旅休闲、本地玩乐等一系列"赛事+"服务消费需求，运动赛事主场也正在成为服务消费主场，"赛事超级周末"效应开始凸显。一张赛事邀请函让消费者享受运动体验快乐，同时感受主场城市的热情与魅力。

以"苏超"和"马拉松"赛事为例，"苏超"比赛期间，美团联动全国超百万商家资源，搭建全国球迷吃喝玩乐一站式服务消费体系。而在全国多地"马拉松"赛事期间，运动健身商家为跑者提供科学备赛服务供给；赛后阶段，美团与本地运动拉伸、理疗放松等商家一起，上线马拉松赛后恢复特色供给，满足跑者赛事全流程服务需求。

《报告》供给侧的数据同样在印证这一趋势。2024至2025年间，美团平台户外运动收录商户由2.7万家增至3.4万家，一年净增超7000家；全国首批49个高质量户外运动目的地中，景区数量新增率达150.9%，户外运动商户新增率达25.2%。

美团相关负责人表示，赛事经济是户外消费扩容的一个典型切口，比如，"马拉松"赛后，"拉伸""按摩"等搜索明显抬升。沿赛道串联周边住宿、餐饮、休闲玩乐商户，把一次性赛事人流转化为可持续的城市消费增量。这种消费趋势的多元发展，折射出户外运动与本地生活消费之间愈发紧密的联动关系。

中国体育用品业联合会运动空间专委会秘书长张庆曾有过类似判断：当流量与真实运动场景相遇，平台加持、技术创新让体育消费更便捷，也让运动空间从"去一次的目的地"沉淀为日常的"生活圈"。《报告》的数据印证了这一方向，消费者走向户外，已不只是为了运动本身，也是在寻找放松、社交、疗愈与重新连接自我的方式，为 "感受"和"意义"付费的意愿持续增强，而户外运动天然具备制造沉浸感、稀缺记忆和传播素材的能力，恰好承接了这一需求的溢出。

北京体育大学体育休闲与旅游学院院长蒋依依表示，2025年我国户外运动消费的发展全貌，为行业精准把握消费趋势、优化产品供给提供了重要参考，也为推动户外运动产业可持续发展、文旅体育深度融合、培育新型消费增长点提供了现实依据。

三、【品牌篇】品牌营销投放前置化、品类差异化，“618”大促正从"集中爆发"走向"全周期心智经营"
1、大促营销节奏整体前移，行业根据自身特性选择转化路径更短的媒介
1.1 2026年“618”典型行业营销投放进一步前置，抢先争夺用户心智

1.2 各行业硬广媒介选择策略持续分化，整体来看品牌倾向于选择离交易更近、转化路径更短的媒介形态
分行业看，美妆护理短视频投放占比提升最明显；食品饮品、服饰箱包加码综合电商，家用电器则显著转向即时通讯媒介，或因家电决策链路长、比价周期久，品牌更需要用即时通讯反复提醒活动、补贴、优惠节点，推动用户下单。


1.3 细分品类投放热度分化明显，广告资源进一步向部分品类集中

2、美妆护理：品牌竞争激烈，注重硬广曝光与直播间引流转化
2.1 美妆护理行业在大促期间仍以硬广曝光为核心手段，通过大规模触达建立品牌认知
细分品类中，家庭护理品同质化高、毛利有限且需要场景实测说服消费者，品牌加大软广种草投放，推高其软广费用占比。

2.2 从细分品类来看，广告资源进一步向基础护肤与彩妆聚集

2.3 美妆行业品牌投放集中度较低，市场格局分散
软广投放头部占比低于硬广，内容种草门槛较低，中小品牌同样有机会争夺话语权。

2.4 面部护肤商品竞争更激烈，头部产品投放费用占比低于香水彩妆，护肤赛道品牌分散、爆品更难形成垄断

2.5 “618”期间美妆品类注重引流至直播间，将硬广曝光转化为品牌私域会员沉淀
营销内容上，面部护肤硬广主打成分功效背书以建立长期信任，香水彩妆硬广侧重妆效体验与限时福利驱动即时转化。

2.6 “618”期间美妆品牌自播占比提升，强化自营直播阵地，从而更好沉淀与转化会员资产

3、食品饮品：软广投放增强，品牌集中度较高，头部品牌主导大促声量
3.1 食品饮品行业仍以硬广投放为主，但愈发注重软广投放
饮料冲调品类在软广投放上更为积极，通过内容触达强化消费场景联想。

3.2 饮料冲调行业软广内容围绕场景化饮用体验展开，强调口感、健康等消费决策关键词

3.3 投放费用仍然高度集中于成熟品牌，老牌企业凭借品牌力和渠道优势持续占据大促投放主导地位

3.4 食品饮品直播销售额集中于白酒、休闲零食和乳品，高客单价与高频消费品类共同拉动行业规模

4、家用电器：品牌竞争从补贴驱动走向多元化，服务与体验成为延伸卖点
4.1 家用电器行业在大促期间兼顾硬软广投放，品类资源分配差异化
生活电器更依赖场景化展示与使用体验说明，软广能够更好承接测评种草、效果演示和生活方式表达，因此相比其他品类，生活电器在软广中投放占比更高。

4.2 空调、冰箱这类大家电仍是投放主力，生活电器、个护小家电品类同步活跃，行业竞争进一步向细分赛道延伸
扫地机器人、电吹风机、电动牙刷等品类进入TOP10，反映出品牌对高频使用、体验型产品的投放意愿增强。

4.3 竞争格局上，家电行业头部品牌营销费用整体领先
美的、海尔等品牌在两类投放上均保持较高占比，而部分品牌更偏单侧发力，体现出品牌在曝光与种草之间的策略分化。

4.4 在营销卖点上，国补相关营销投放虽然同比减少，但仍是家用电器行业重要的促销卖点
与此同时，安装服务、售后保障等体验型卖点成为新的用户吸引点，品牌竞争从价格向服务延伸。

5、服饰箱包：品牌从抢曝光进一步走向种草与转化联动
5.1 2026年“618”服饰箱包行业更注重软硬协同，同时品类存在差异
奢侈品更重品牌形象、稀缺感和官方叙事，所以注重硬广做高端形象曝光；男女服饰、鞋履、内衣袜子则更贴近日常消费，用户需要更多内容种草和穿搭说服，因此相比其他品类在软广中更活跃。

5.2 服饰箱包行业硬广与软广投放节奏形成联动，在营销链路中承担不同角色
其中，硬广更侧重在大促节点抢占曝光与导流转化，软广则通过更连续的内容投放维持种草热度并延长消费决策链路影响。

5.3 不同细分品类的品牌集中度存在差异，头部品牌在硬广和软广投放上均有明显优势
奢侈品虽以硬广为基本盘，但部分品牌如卡地亚、拉夫劳伦在软广布局上相比其他品牌更积极。

5.4 服饰箱包行业直播销售规模可观，品牌官方直播间是重要流量转化阵地
